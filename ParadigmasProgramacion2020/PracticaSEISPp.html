Práctica 6

Temario

CONSTRUCCIÓN DE CLASES Y OBJETOS

Objetivo

Construirá clases y objetos en el paradigma orientado a objetos.


Actividades


Construir las clases: FiguraGeometrica, Circulo, Triangulo y Poligono.


Cuestionario


Pregunta 1. Para cada una de las clases construir los métodos para calcular el área y el perimetro.


Pregunta 2. Construir el método principal en una clase diferente a las creadas.


Pregunta 3. Construir el método principal en una clase de las construidas..


Pregunta 4. Construir los objetos para cada una de las clases para calcular el área.


Pregunta 5. Construir los objetos para cada una de las clases para calcular el perimetro.


Pregunta 6. Llamar al objeto cuando esta en una clase principal diferente a las clases construidas.


Pregunta 7. Llamar al objeto cuando esta en una clase principal igual a una de las construidas.


Pregunta 8. Declarar la variable radio como static en la clase Circulo y llamarla desde la clase principal. Mostrar las formas como se puede llamar dicha variable.
