1.2.3 FUNCIONES PURAS, DE PRIMER ORDEN Y DE ORDEN SUPERIOR

Funciones puras


  • Paradigma imperativo. Es como se hace
  • Paradigma declarativo. Es que quiere.
  • Paradigma orientado a objetos. Es modelar conceptos como objetos interactuantes.
  • Paradigma funcional. Es componer transformaciones puras evitando compartir estados.


    • La notación para lambda es la siguiente: 


         lambda (parámetro) (expresión)
     
     λx.M
     
     lambda (x) (x*x)
     
     cuadrdado(lambda (x) (x*x)) 5
     
     (λx.x*x)5

    Función pura. dónde la misma entrada produce la misma salida. No tiene efectos colaterales que incluyen cosas que modifiquen la variable global.


    Funciones primer orden


    Funciones de primer orden. son funciones que se pueden tratar como otros valores, tales como un número, una cadena, o una lista. Se puede asignarlas a variables, pasarlas como argumentos a otras funciones, regresarlas como el resultado de otras funciones.


    Asignar una función a una variable:


         def decirHola(nombre):
         return f"Hola,{nombre}"
     
     saludo = decirHola
     
     print(saludo("Alicia"))

    Pasar una función como argumento:


         def operarLista(listaDatos,operatcion):
         return[operacion(elemento) for elemento in listaDatos]

     def hacerMayusculas(texto):
         return texto.upper()

     def agregarEntusiasmo(texto):
         return f"{texto}"

     miCadena = ["python","es","flexible"]

     print(operarLista(miCadena,hacerMayuscula))

     print(operarLista(miCadena,agregarEntusiasmo))

    Returnar una función de otra función:


         def crearMultiplicador(factor):
         def multiplicador(numero):
             return numero * factor
         return multiplicador

     doble = crearMultiplicador(2)

     triple = crearMultiplicador(3)

     print(doble(10))
     print(triple(10))

    Transparencia referencial: significa reeplazar una función por su valor resultante sin alterar el comportamiento de un programa.


    Se tiene una función pura: 


         def calculoImpuesto(precio,impuesto):
         return precio * (1 + impuesto)

    Se calcula el total: 


         subTotal = 100
     envio = 10.0
     impuesto = 0.07

     ordenTotal = calculoImpuesto(subtotal,impuesto) + envio

     print(f"El total es: ${ordenTotal:.2f}")

    El siguiente programa es solo:


         subTotal = 100
     envio = 10.0
     impuesto = 0.07

     ordenTotal = 107.0 + envio
     print(f"El total es: ${ordenTotal:.2f}")

    Programa con una función impura: 


         IMPUEST_MINIMO = 0.05

     def calculoImpuesto(precio):
         precioTotal = precio * (1 + IMPUEST_MINIMO)
         print(f"alculo del precio final, es:{precioTotal:.2f}")
         return precioTotal

     subTotal = 100
     envio = 10.0

     ordenTotalI = calcularImpuesto(subtotal) + envio
     print(f"El total es: ${ordenTotalI:.2f}")

    Reeemplazo a la función impura: 


         subTotal = 100
     envio = 10.0

     ordenTotalI = 105.0 + envio

     print(f"El total es: ${ordenTotalI:.2f}")

    Si lo hacemos por medio de valores: 


         precioFinal = (precioInicial - descuento) * (1 + impuesto) + envio

     # Si las variables son valores, se pueden extraer de la función

     def precioFinal(precioInicial,descuento,impuesto,envio):
         return (precioInicial - descuento) * (1 + impuesto) + envio

     # Se puede reemplazar la línea original por

     precioFinal = calculoPrecio(precioInicial,descuento,impuesto,envio)
