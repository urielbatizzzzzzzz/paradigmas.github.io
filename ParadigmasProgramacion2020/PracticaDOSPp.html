Práctica 2

Temario

RECURSIÓN EN PROGRAMACIÓN FUNCIONAL

Objetivo

Se crearan funciones utilizando la recursividad.


Actividades


Crear la función, donde se utilice la recursividad par los siguientes problemas:


Problema 1: Presentar un programa que, imprima la suma de los números indicados.


Problema 2: Presentar un programa que, imprima los números que se soliciten.


Problema 3: Presentar un programa que, imprima el abecedario.


Cuestionario


Pregunta 1. Construir los programas 1, 2 y 3 en Python.


Pregunta 2. Construir un programa en Python que, haga la multiplicación de los siguientes polinomios: f1(x) = 3x3 + 4x + 7 y f2(x) = 7x2 + 3x +50.


Pregunta 3. Construir un programa en Python que, haga la composición de los siguientes polinomios: f(x) = 3x3 + 4x + 7 y g(x) = 7x2 + 3x +50..


Pregunta 4. Construir un programa en Python que, haga el encadenamiento de los siguientes polinomios: f(x) = 3x3 + 4x + 7 , g(x) = 7x2 + 3x +50 y h(x) = 7x + 12.


Pregunta 5. Construir un programa en C++ que calcule el factorial de un número n dado por teclado.


Pregunta 6. Construir un programa en Python que calcule el factorial de un número n dado por teclado.


Pregunta 7. Construir un programa en C++ y también en Python que reciba desde teclado n que representa el número de discos y que presente en consola todos los pasos a seguir para mover los discos del poste A al poste C y utilizar el poste Bde comodino.
